# Automação de Tickets com Python

## 📌 Sobre o projeto
Script desenvolvido para simular a organização e priorização de chamados de suporte técnico.

## ⚙️ Funcionalidades
- Classificação de tickets por prioridade (alta, média, baixa)
- Identificação de tickets urgentes
- Geração de relatório por tipo de incidente
- Exportação de dados em CSV

## 🛠️ Tecnologias utilizadas
- Python
- Manipulação de listas e dicionários
- Biblioteca CSV

## 📊 Exemplo de saída
- Tickets urgentes exibidos no terminal
- Relatório gerado automaticamente (`relatorio.csv`)


## 🎯 Objetivo
Simular um cenário real de suporte técnico com automação de tarefas, organização de chamados e geração de relatórios.
